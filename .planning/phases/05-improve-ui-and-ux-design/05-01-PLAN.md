Open `static/style.css` and make these 6 changes:

**P1 — Contrast fix (line ~23):**
Change `--color-text-faint: #555555;` to `--color-text-faint: #686868;`

**P2 — Focus-visible on buttons (after .btn-destructive:hover block, ~line 240):**
Add this rule block:
```css
.btn-cta:focus-visible,
.btn-secondary:focus-visible,
.btn-destructive:focus-visible {
  outline: none;
  box-shadow: 0 0 0 2px var(--color-accent);
}
```

**P4 — Touch targets (in .btn-cta block, ~line 197):**
Change `min-height: 40px;` to `min-height: 44px;`
Add `min-height: 44px;` to `.btn-secondary` block (after `display: inline-flex; align-items: center;` line).
Add `min-height: 44px;` to `.btn-destructive` block (after `display: inline-flex; align-items: center;` line).

**P7 — Replace hardcoded 12px with tokens:**
- `.card-grid { gap: 12px; }` (~line 262) -> `gap: var(--space-sm);` (8px — tighter grid is intentional for card layouts)
- `.card-body { padding: 12px; }` (~line 285) -> `padding: var(--space-sm) var(--space-md);` (8px top/bottom, 16px left/right — cards need more horizontal breathing room)
- `.form-input { padding: var(--space-sm) 12px; }` (~line 300) -> `padding: var(--space-sm) var(--space-md);`
- `.toast { padding: 12px 20px; }` (~line 515) -> `padding: var(--space-sm) var(--space-lg);` (8px/24px maps to existing tokens)
- `.toast { gap: 12px; }` (~line 519) -> `gap: var(--space-sm);`

**P8 — Active state on buttons (after the new focus-visible block):**
Add this rule block:
```css
.btn-cta:active,
.btn-secondary:active,
.btn-destructive:active {
  transform: scale(0.97);
}
```

**P9 — Card hover shadow fix (~line 254):**
Change `.card:hover { box-shadow: 0 8px 32px rgba(0, 0, 0, 0.6); }` to:
```css
.card:hover {
  transform: translateY(-2px);
  box-shadow: 0 0 0 1px rgba(255,255,255,0.1), 0 8px 24px rgba(0, 5, 20, 0.8);
}
```
