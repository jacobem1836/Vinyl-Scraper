Two changes in `static/typeahead.js`:

**1. Add spinner hide to `closeDropdown()` (per D-01, root cause 1):**

In the `closeDropdown(prefix)` function (currently lines 31-39), destructure `spinner` from `getEls(prefix)` and add `spinner.classList.add("hidden")`. The function currently only destructures `input` and `listbox`. Change:

```javascript
function closeDropdown(prefix) {
    const { input, listbox, spinner } = getEls(prefix);
    if (listbox) listbox.classList.remove("typeahead-dropdown--open");
    if (input) {
      input.setAttribute("aria-expanded", "false");
      input.removeAttribute("aria-activedescendant");
    }
    if (spinner) spinner.classList.add("hidden");
    if (state[prefix]) state[prefix].activeIndex = -1;
  }
```

This ensures the spinner is hidden on every path that calls `closeDropdown()` — Escape, Tab, blur, and any future close path. The explicit `spinner.classList.add("hidden")` calls in `selectResult()` and the type change handler become redundant safety nets (leave them in place).

**2. Cancel debounce timer on result select and type change (per D-01, root cause 2):**

In `selectResult()` (currently line 156), add debounce cancellation at the start of the function body, before `closeDropdown(prefix)`:

```javascript
window.selectResult = function (prefix, result) {
    if (state[prefix] && state[prefix].timer) {
      clearTimeout(state[prefix].timer);
      state[prefix].timer = null;
    }
    const { input, hidden, spinner } = getEls(prefix);
    // ... rest unchanged
```

In the type change handler (currently line 274), add debounce cancellation at the start of the handler, before `closeDropdown(prefix)`:

```javascript
typeSelectEl.addEventListener("change", function () {
        if (state[prefix] && state[prefix].timer) {
          clearTimeout(state[prefix].timer);
          state[prefix].timer = null;
        }
        closeDropdown(prefix);
        // ... rest unchanged
```

This prevents the race condition where: user types -> debounce starts -> user selects result -> spinner hidden -> debounce fires -> spinner shown again with no pending fetch to hide it.
