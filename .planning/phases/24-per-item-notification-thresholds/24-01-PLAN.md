<objective>
Add per-item notification threshold support so each wishlist item can override the global default.

Purpose: Users want different alert sensitivity per item -- a rare pressing might warrant a lower threshold than a common album. Currently all items share a single hardcoded 20% threshold.

Output: Nullable `notify_below_pct` column, global default in Settings, fallback logic in notifier, updated edit form and item detail display.
</objective>
