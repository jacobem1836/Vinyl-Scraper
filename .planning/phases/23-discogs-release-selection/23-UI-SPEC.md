## Interaction Contract

### Pin Modal Open/Close

- Trigger: "Pin Release" button (`btn-secondary`) in item header action row
- Open sequence: remove `.hidden` → `requestAnimationFrame` → remove `opacity-0` from backdrop → remove `opacity-0 translate-y-4` from panel → focus first input
- Close sequence: add `opacity-0` to backdrop → add `opacity-0 translate-y-4` to panel → `setTimeout 200ms` → add `.hidden` → return focus to trigger button
- Backdrop click closes modal (match existing pattern)
- Escape key closes modal (match existing base.html pattern)

### Release Search

- Input field inside modal, placeholder: "Search Discogs releases..."
- Debounce: 300ms (match existing typeahead timing from `base.html`)
- Minimum query length: 2 characters before firing
- Results: up to 10 rows, scrollable if overflow
- Spinner: `typeahead-spinner spinner` class shown during fetch, hidden on result or error
- Each result row: 40×40 thumbnail (left, `object-fit: cover`) + right column (`Artist — Title` at 16px, Year at 14px muted). Full row is a clickable button.
- Clicking a row: marks it as selected (accent left-border highlight), does not immediately save

### Pin Confirm

- After selecting a row, a "Pin Release" confirm button appears in the modal footer (or activates if already rendered disabled)
- Confirm submits `POST /wishlist/{item_id}/pin-release` with `release_id` + `artwork_url`
- On success: modal closes, page reloads (or redirects with toast) showing updated artwork and pin status label
- Cancel button (`btn-secondary`): closes modal without saving

### Clear Pin

- "Clear pin" link/button (`btn-destructive` style) inside modal — visible only when item already has a pinned release
- No secondary confirmation dialog — single click clears and submits immediately (the "Cancel" path in the modal serves as the implicit undo)
- Submits `POST /wishlist/{item_id}/pin-release` with `release_id: null`

### Pin Status Display

- Shown below 120×120 artwork block on item detail page
- Format: `Pinned: [Title] ([Year])`
- Style: 12px, `--color-text-muted`, no badge or border — plain text line
- Absent entirely when nothing is pinned (no placeholder or empty state needed here)
