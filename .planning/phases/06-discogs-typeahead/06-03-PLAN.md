<task type="auto">
  <name>Task 1: Hide spinner in selectResult() on selection</name>
  <files>static/typeahead.js</files>
  <action>
In the `window.selectResult` function (line 155), add `spinner` to the getEls() destructure. After calling `closeDropdown(prefix)`, add one line to hide the spinner.

Current code (lines 155–167):
```
window.selectResult = function (prefix, result) {
  const { input, hidden } = getEls(prefix);
  // ...
  closeDropdown(prefix);
  if (input) input.focus();
};
```

New code:
```
window.selectResult = function (prefix, result) {
  const { input, hidden, spinner } = getEls(prefix);
  if (input) input.value = result.title;
  if (hidden) {
    hidden.value = result.release_id || "";
  }
  closeDropdown(prefix);
  if (spinner) spinner.classList.add("hidden");
  if (input) input.focus();
};
```

This closes gaps from UAT tests 3 & 4: "loading icon stays" after selection. The spinner is now explicitly hidden when a result is selected.

  </verify>

In the type change event listener (lines 271–278), add `spinner` to the getEls() destructure. After calling `closeDropdown(prefix)`, add one line to hide the spinner.

Current code (lines 271–278):
```
if (typeSelectEl) {
  typeSelectEl.addEventListener("change", function () {
    closeDropdown(prefix);
    const { hidden } = getEls(prefix);
    if (hidden) hidden.value = "";
  });
}
```

New code:
```
if (typeSelectEl) {
  typeSelectEl.addEventListener("change", function () {
    closeDropdown(prefix);
    const { hidden, spinner } = getEls(prefix);

    if (spinner) spinner.classList.add("hidden");
  });
}
```

This closes the gap from UAT test 6: "loading icon still stays" when switching to Key Word/s type. Spinner is now hidden when the user changes type, regardless of whether the new type has typeahead enabled.
