- MSO conditional comments wrapping Outlook fallback styles: `<!--[if mso]>` block that sets body background to white and text to dark (#333333) for Word-rendering clients that strip background colors

**Color palette (per D-03) — ALL as inline hex, no CSS custom properties:**
- Body background: #0a0a0a
- Content area / card background: #111111
- Primary text: #f5f5f5
- Accent / headings: #ffffff
- Borders: #222222
- Muted text: #777777

**Hero summary block (per D-01, D-05):**
- Top section of email with CRATE text wordmark header (plain text "CRATE" in white, letter-spacing: 6px, font-size: 12px, uppercase)
- Item name as heading: `{{ item_name }}` in white (#ffffff), font-size 24px, font-weight bold
- Type badge next to or below item name: `{{ item_type }}` — styled as inline text with muted color (#777777), uppercase, font-size 11px, letter-spacing 1px
- Best landed price: `{{ best_landed_price }}` in large white text, font-size 32px, font-weight bold
- Percentage below typical: `{% if has_typical_price %}{{ pct_below_typical }} below typical{% endif %}` — displayed in #34d399 (success green), font-size 14px
- Notification threshold note: `Notifying on listings ≥ {{ notify_below_pct }} below typical price` in muted text (#777777), font-size 12px

**Listing table (per D-06):**
- Table with columns: Title, Landed Price (AUD), Source, Ships From
- Table header row: background #0a0a0a, text #777777, uppercase, font-size 11px, letter-spacing 1px
- Table body rows: background #111111, text #f5f5f5, font-size 14px
- Row borders: 1px solid #222222
- Cell padding: 12px
- Jinja2 loop: `{% for listing in listings %}`
- Each row: `{{ listing.title }}`, `{{ listing.landed_price }}`, `{{ listing.source }}`, `{{ listing.ships_from }}`
- No per-listing links (per D-02)

**Footer (per D-02):**
- Single centered link: "View on CRATE" pointing to `{{ item_url }}`
- Link styled as white text on transparent background, font-size 14px, text-decoration underline
- Below that: muted text "You're receiving this because a listing matched your deal alert threshold." in #777777, font-size 11px
- No unsubscribe link (personal single-user tool)

**Email client compatibility rules:**
- ALL styles inline on elements (no `<style>` block — Gmail strips `<style>` in non-AMP emails)
- Use `<table>`, `<tr>`, `<td>` for layout — no divs for structure
- Use `align="center"` attributes on tables
- Use `cellpadding` and `cellspacing` attributes
- Use `width` attributes (not just CSS width)
- Use `border-collapse:collapse` on tables
- Font stack: `font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Helvetica, Arial, sans-serif;`
- All images (if any) have alt text
- No CSS custom properties (var(--x))
- No flexbox or grid
- No shorthand `background` — use `background-color` explicitly
