**Skeleton loading state** (D-16):

1. In `static/style.css`, add skeleton keyframes and classes:
   ```css
   /* Image loading skeleton (D-16) */
   @keyframes skeleton-pulse {
     0% { background-position: 200% 0; }
     100% { background-position: -200% 0; }
   }
   
   .card-artwork-wrapper {
     position: relative;
     aspect-ratio: 1 / 1;
     width: 100%;
     background: linear-gradient(90deg, #1a1a1a 25%, #2a2a2a 50%, #1a1a1a 75%);
     background-size: 200% 100%;
     animation: skeleton-pulse 1.2s ease-in-out infinite;
   }
   
   .card-artwork-wrapper img {
     opacity: 0;
     transition: opacity 200ms ease-out;
   }
   
   .card-artwork-wrapper img.loaded {
     opacity: 1;
   }
   
   .card-artwork-wrapper.loaded {
     animation: none;
     background: transparent;
   }
   ```

2. In `templates/index.html`, wrap each card's `<img class="card-artwork">` in a `<div class="card-artwork-wrapper">`. Update the `<img>` tag to NOT have `class="card-artwork"` on the img itself (move `aspect-ratio` and `width` to the wrapper). Add `onload="this.classList.add('loaded'); this.parentElement.classList.add('loaded');"` to each `<img>`.

   Before (line ~154-167):
   ```html
   {% if item.artwork_url %}
     <img class="card-artwork" src="..." ...>
   {% else %}
     <img class="card-artwork" src="/static/vinyl-placeholder.svg" ...>
   {% endif %}
   ```

   After:
   ```html
   <div class="card-artwork-wrapper">
   {% if item.artwork_url %}
     <img class="card-artwork" src="/api/artwork?url={{ item.artwork_url | urlencode }}" alt="{{ item.query }} album cover" onerror="this.src='/static/vinyl-placeholder.svg'; this.classList.add('loaded'); this.parentElement.classList.add('loaded');" onload="this.classList.add('loaded'); this.parentElement.classList.add('loaded');" loading="lazy">
   {% else %}
     <img class="card-artwork" src="/static/empty-vinyl-placeholder.png" alt="{{ item.query }} -- no artwork available" onload="this.classList.add('loaded'); this.parentElement.classList.add('loaded');">
   {% endif %}
   </div>
   ```

3. In `templates/item_detail.html`, apply the same skeleton treatment to the 120x120 artwork thumbnail (lines ~15-28). Wrap in a `<div class="card-artwork-wrapper" style="width: 120px; height: 120px;">` and add `onload`/`onerror` handlers.

**Post-add image refresh** (D-16):

In `templates/index.html`, in the scanning-item polling block (lines ~259-281), when `data.has_listings` is true AND `data.artwork_url` is provided, update the card's image src directly instead of reloading the page:
```javascript
if (data.has_listings) {
  clearInterval(timer);
  if (data.artwork_url) {
    const img = card.querySelector('img');
    if (img) {
      img.src = '/api/artwork?url=' + encodeURIComponent(data.artwork_url) + '&t=' + Date.now();
    }
  }
  window.location.reload();
}
```

Note: The page reload is kept as fallback since the price data also needs to update. The image cache-bust (`?t=timestamp`) ensures the new artwork loads without browser cache issues. This is a practical improvement — the full fix would require partial DOM updates which is out of scope.
