**P6 — 3-column grid breakpoint:**
Add a new `@media` rule between the existing 768px (2-col) and 1280px (4-col) breakpoints. Insert it after the `@media (min-width: 768px)` block and before the `@media (min-width: 1280px)` block:

```css
@media (min-width: 1024px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

The final breakpoint progression should be: 1-col (default) -> 2-col (768px) -> 3-col (1024px) -> 4-col (1280px).
